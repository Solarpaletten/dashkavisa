## Текущая версия

### v0.1 — Initial Internal Release ✅

**Статус:** Зафиксирован, режим сопровождения

**Что включено:**
- Telegram-бот с базовыми командами
- Авторизация в VFS Global
- Проверка доступных слотов
- Система уведомлений
- Базовое логирование

**Известные ограничения:**
- Cloudflare может блокировать автоматизацию
- Single-user режим
- Нет автоматического бронирования

## Что НЕ планируется

Следующие функции **не входят** в roadmap любой версии:

- Публичный SaaS-сервис
- Мобильное приложение
- Поддержка других визовых систем (не VFS)
- Интеграция с платёжными системами
- Автоматическое заполнение анкет

---

## Принципы развития

### 1. Стабильность важнее фич

Новые функции добавляются только после стабилизации текущих.

### 2. Минимальные изменения

Каждая версия — минимальный набор изменений для достижения цели.

### 3. Обратная совместимость

Изменения не должны ломать существующую конфигурацию.

### 4. Документация первична

Любое изменение сопровождается обновлением документации.

```
1. Идея/проблема

      ↓
5. Приоритизация
      ↓
6. Реализация
      ↓
7. Документация
      ↓
8. Релиз
```
